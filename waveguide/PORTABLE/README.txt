Instructions to run the waveguide patch from any MAC

Extra software
==============

Install these programs (they are freely available, google them)

* Supercollider 3.6 or higher
* Reaper 
* JackPilot

* Run the file "install.sh"
* A folder "waveguide" will be created in your desktop. Leave it there
* Open that folder and go to the subfolder SC
* There you will see a .scd file (something like patchxx.scd)
* Launch Supercollider and open that file
* Look for further instructions inside the patch
